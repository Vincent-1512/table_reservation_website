   - 8 tables, nhiều relationships

## 📝 CHEAT SHEET - 10 ĐIỀU PHẢI NHỚ

1. **Project:** Restaurant - Web đặt bàn & menu điện tử
2. **Framework:** Spring Boot 4.0.6
3. **Database:** MySQL 8.0 (Hibernate ORM)
4. **Security:** Spring Security + BCrypt
5. **Views:** Thymeleaf (template engine)
6. **Entities:** 8 (User, Order, MenuItem, Reservation, DiningTable, Category, Area, OrderItem)
7. **Roles:** ADMIN, STAFF, CUSTOMER
8. **Cart:** @SessionScope (lưu trong session)
9. **Race Condition:** @Version (Optimistic Locking)
10. **Run:** `docker-compose up --build` hoặc `./mvnw spring-boot:run`
